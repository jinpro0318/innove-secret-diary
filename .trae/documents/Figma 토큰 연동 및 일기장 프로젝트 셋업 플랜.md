# Figma 토큰 기반의 디자인 시스템 구축 플랜

사용자님이 원하시는 **"나만의 감성이 담긴 시크릿 일기장"**을 구현하기 위해, 디자인과 코드가 완벽하게 동기화되는 개발 환경을 구축해 드리겠습니다. Figma의 디자인 토큰을 코드(Tailwind CSS)로 자동 변환하는 파이프라인을 설정하여, 디자인 변경 사항이 즉시 앱/웹에 반영되도록 합니다.

## 1. 프로젝트 초기화 (Next.js + Tailwind CSS)
웹사이트와 앱(PWA/Webview)을 동시에 배포하기 가장 유리한 **Next.js** 프레임워크를 기반으로 시작합니다.
*   **Next.js 14**: 최신 React 기능 활용
*   **Tailwind CSS**: 디자인 토큰을 쉽게 적용할 수 있는 유틸리티 CSS
*   **TypeScript**: 안정적인 개발 환경

## 2. 디자인 토큰 파이프라인 구축 (Figma -> Code)
Figma에서 정의한 토큰(색상, 폰트, 간격 등)을 코드로 변환하는 자동화 시스템을 설치합니다.
*   **Style Dictionary 설치**: JSON 형태의 토큰 데이터를 CSS 변수로 변환해주는 표준 도구입니다.
*   **토큰 구조 설계**: `tokens/design-tokens.json` 파일을 생성하여 Figma에서 추출될 데이터의 구조를 잡습니다.
*   **변환 스크립트 작성**: `npm run tokens` 명령어 한 번으로 디자인 변경 사항이 CSS에 반영되도록 설정합니다.

## 3. Tailwind CSS와 토큰 연결
변환된 CSS 변수들을 Tailwind 설정(`tailwind.config.ts`)에 매핑하여, 개발 시 직관적인 클래스명(예: `bg-brand-primary`, `text-body-large`)으로 사용할 수 있게 합니다.

## 4. (옵션) 핸드메이드 감성 UI 컨셉 제안
구축된 시스템 위에서 실제로 구현할 "핸드메이드 일기장"의 디자인 컨셉이 필요하시다면, 저의 전문 에이전트(`handmade-diary-designer`)를 통해 종이 질감, 손글씨 폰트, 아날로그 감성의 UI 아이디어를 구체화해 드릴 수 있습니다.
