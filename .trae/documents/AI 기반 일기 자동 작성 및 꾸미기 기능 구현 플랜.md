# 구글 AI (Gemini) 기반 일기장 기능 확장 플랜

사용자님이 제공해주신 구글 API 키(`AIza...`)를 활용하여, "Secret Handmade Diary"에 **AI 글쓰기 도우미**와 **자동 꾸미기** 기능을 탑재하겠습니다. "Google Nano Banana"는 구글의 최신 AI 모델인 **Gemini** (특히 빠르고 효율적인 Gemini Flash/Pro)를 의미하는 것으로 판단하여 이를 적용합니다.

## 1. 환경 설정 및 보안
*   **API 키 보안**: 제공해주신 API 키는 `.env.local` 파일에 안전하게 저장하여 코드에 직접 노출되지 않도록 합니다.
*   **라이브러리 설치**: Google의 공식 SDK인 `@google/generative-ai`를 설치합니다.

## 2. 기능 1: AI 일기 다듬기 (Refine Entry)
음성이나 짧은 메모를 완벽한 일기로 변환해주는 기능입니다.
*   **음성 인식 (Voice to Text)**: 브라우저 내장 Web Speech API를 사용하여 마이크 버튼을 누르면 사용자의 말을 텍스트로 변환합니다.
*   **AI 재작성 (Rewrite)**: 변환된 텍스트를 Gemini에게 전송하여, 현재 다이어리의 감성(따뜻함, 서정적)에 맞는 문체로 자연스럽게 이어줍니다.

## 3. 기능 2: 감성 맞춤 자동 꾸미기 (Auto Decorate)
일기 내용을 분석하여 그 날의 기분에 맞는 스티커와 배경 분위기를 자동으로 적용합니다.
*   **감정 분석**: Gemini가 일기 내용을 읽고 '기쁨', '우울', '평온' 등의 감정을 파악합니다.
*   **스티커 배치**: 분석된 감정에 어울리는 이모지(스티커 역할)나 아이콘을 JSON 형태로 받아와, 일기장 위에 자연스럽게 흩뿌려진 듯 배치합니다. (예: 우울함 -> ☁️, 💧 / 기쁨 -> 🌻, ✨)
*   **배경 틴트**: 종이 배경의 색감을 미세하게 조절하여 분위기를 더합니다.

## 4. UI/UX 업데이트
*   **마이크 버튼**: 아날로그 감성에 맞는 마이크 아이콘 버튼 추가.
*   **매직 완드 버튼**: AI 기능을 실행하는 "마법 지팡이" 또는 "깃펜" 모양의 버튼 추가.
*   **결과 화면**: AI가 꾸며준 일기를 즉시 미리보기 할 수 있는 인터랙션 구현.

**이 플랜을 승인하시면:**
1.  `.env.local` 파일을 생성하여 API 키를 설정합니다.
2.  `@google/generative-ai`를 설치하고 백엔드 API를 구현합니다.
3.  프론트엔드에 마이크 및 AI 버튼 기능을 추가하여 완성하겠습니다.
